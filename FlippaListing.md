# ResiliNet — Community Climate Resilience App (React Native + Firebase)

**Price (asking):** [Set after metrics]

## Short summary
ResiliNet is a cross-platform (iOS/Android) community resilience app that delivers local hazard alerts, a help board for offers/requests, and a map of verified safe locations. Built with React Native (Expo) and Firebase. Includes a small admin web app to create verified alerts/POIs. Ideal for civic-tech buyers, municipalities, NGOs, or impact-focused acquirers.

- Full mobile source code (React Native / Expo)

- Deployment scripts and README with Firebase setup
- App store assets: app name, icon, long/short descriptions, screenshots placeholders
- Firestore rules and basic security setup
- 30-day handover support (technical guide & onboarding doc)
- Pitch deck and marketplace listing copy included

## Key features (MVP)
- Local feed of alerts (admin-posted + public feeds)
- Interactive map with verified shelters / POIs
- Help board to post requests and offers (food, shelter, volunteers)
- Simple sign-in (Email/Password via Firebase)
- Admin dashboard to publish verified alerts and POIs

## Reason to buy

## How to run (for buyer)
1. Create Firebase project; enable Firestore and Auth.
2. Update `firebaseConfig.js` with Firebase keys.
3. `npm install` and `npm run build` or `expo publish`.
4. Optionally integrate FCM for push alerts & Stripe for org billing.
